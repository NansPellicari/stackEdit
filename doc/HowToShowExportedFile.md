
# Steps

 1. Create a file
 2. add a gist publish export
 3. publish
 4. go to the gist url (click on the gist button in the top bar)
 5. retrieve the url and replace `gist.github.com` by `gist.nans.me` and add `/raw` to the end.
 6. That's it
